🎸 Day 01 – Band Name Generator

This quirky little program helps you generate a band name based on your hometown and your pet’s name. It’s a fun way to get started with Python input/output and string formatting!

💡 How It Works

  1. User is greeted with a welcome message.
  2. They're asked which city they grew up in.
  3. They're asked for the name of their pet.
  4. The program combines the two to suggest a band name.

🧪 Try It Yourself

This demo is a static preview of the project’s logic.


👋 Greetings, Stranger!

🐶 What is the name of your pet? → Neko
