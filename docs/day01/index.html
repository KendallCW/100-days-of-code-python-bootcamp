🎸 Day 01 – Band Name Generator

This playful project was the very first step into Python. It asks a couple of questions—your city and your pet’s name—and blends them together to suggest a band name. A great way to explore user input and string manipulation in a fun way!

💡 How It Works

  1. It greets the user with a welcome message.
  2. Asks which city they grew up in.
  3. Then it asks for the name of their pet.
  4. Finally, it combines both to reveal a custom band name.
